# PIE Portfolio Analysis

This project adds a Streamlit dashboard on top of the snapshot files generated by `data_retrival.py`.

## Run the dashboard

```bash
streamlit run app.py
```

## What it does

- Scans `./data/` for available snapshot dates
- Loads the selected `PIE_*` exposure files plus the combined holdings parquet
- Shows overview exposure charts, concentration metrics, company search, country and sector drilldowns, and ETF overlap analysis

## Notes

- No command-line interface is required for the report builder or dashboard
- `data_retrival.py` is left unchanged
